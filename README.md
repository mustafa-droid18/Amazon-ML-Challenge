## Setting Up Virtual Env
1. Windows 
```

    pip install virtualenv 
    virtualenv env
    Set-ExecutionPolicy Unrestricted -Scope Process
    # In cmd.exe
    env\Scripts\activate.bat
    # In PowerShell
    env\Scripts\activate.ps1
```
